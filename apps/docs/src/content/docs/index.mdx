## Packages

| Package | Description |
|---|---|
| [`@bpmn-sdk/core`](/packages/core/) | Fluent builder, parser/serializer, auto-layout, AI-compact format |
| [`@bpmn-sdk/engine`](/packages/engine/) | Zero-dep BPMN simulation engine (browser + Node.js) |
| [`@bpmn-sdk/api`](/packages/api/) | Camunda 8 REST API client — 180 typed methods |
| [`@bpmn-sdk/canvas`](/packages/canvas/) | SVG BPMN viewer with pan/zoom, dark/light theme |
| [`@bpmn-sdk/editor`](/packages/editor/) | Full BPMN editor — canvas + properties panel + AI bridge |
| [`casen` (CLI)](/cli/casen/) | Interactive TUI for managing Camunda 8 from the terminal |

## Quick Example

```typescript
import { Bpmn } from "@bpmn-sdk/core";

const xml = Bpmn.export(
  Bpmn.createProcess("approval-flow")
    .startEvent("start", { name: "Request Submitted" })
    .userTask("review", { name: "Review Request" })
    .exclusiveGateway("gw", { name: "Approved?" })
    .branch("yes", (b) =>
      b.condition("= approved")
        .serviceTask("notify", { taskType: "send-email" })
        .endEvent("done")
    )
    .branch("no", (b) =>
      b.defaultFlow().endEvent("rejected")
    )
    .withAutoLayout()
    .build()
);
// → valid BPMN 2.0 XML, auto-laid-out, Zeebe-ready
```
